initialize.js

Organize your JS doing Components! Examples: #{Components: Yellow and Opacity} and #{Component: Blue}.

  1. import javascript
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.0/jquery.min.js"></script>
  1. create component and everytime, you will need to reference to your element
function MyComponent(el) {
  $(el).append('<span>Custom Component Append</span>');
}
  1. set data-attribute, if you want to have more components in the same element, you can! see the examples on this repository
  <div data-initialize="MyComponent">something will happen here</div>
